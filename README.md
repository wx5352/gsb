# AI卖货教学网站 - 老年人专属课程

这是一个专为老年人设计的AI卖货教学平台，帮助老年朋友轻松学会使用AI工具进行卖货。

## 项目特点

- 🎯 **专为老年人设计**：大字体、清晰布局、简单操作
- 📚 **系统化课程**：从基础到进阶，循序渐进
- 🎥 **视频教学**：真人演示，一看就会
- 📱 **响应式设计**：支持电脑、平板、手机访问
- 💡 **实用案例**：真实案例分享，即学即用

## 课程内容

1. **第一课：什么是AI卖货？** - 了解AI卖货的基本概念
2. **第二课：选择合适的AI工具** - 学习如何选择适合的工具
3. **第三课：注册和登录AI平台** - 手把手教您注册账号
4. **第四课：使用AI生成商品描述** - 快速生成吸引人的商品描述


## 使用方法

### 本地运行

1. 下载或克隆项目文件
2. 直接在浏览器中打开 `index.html` 文件即可

### 使用本地服务器（推荐）

```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js的http-server
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 自定义视频

当前使用的是示例视频链接。要使用您自己的视频：

1. 将视频文件放在项目目录中（或使用在线视频链接）
2. 编辑 `script.js` 文件中的 `courses` 数组
3. 将 `videoUrl` 替换为您的视频路径或链接

示例：
```javascript
{
    id: 1,
    title: "第一课：什么是AI卖货？",
    videoUrl: "videos/lesson1.mp4",  // 本地视频
    // 或
    videoUrl: "https://your-video-url.com/video.mp4",  // 在线视频
    ...
}
```
